# Gold Production Historical Database

Historical gold mine production data from 1493 to 2025, compiled from multiple sources.

## Overview

- Period: 1493-2025
- Countries: 141
- Observations: ~550
- Unit: Kilograms (kg)

## Data Sources

| Source | Period | Priority |
|--------|--------|----------|
| Ridgway (1929) | 1871-1927 | 5 |
| Ridgway (1929) - Decadal | 1801-1861 | 3 |

| BGS (British Geological Survey) | 1971-2014 | 6 |
